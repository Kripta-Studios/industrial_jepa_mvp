# Claims and Limitations Matrix

This document lists the technical claims that can be made about the `industrial_jepa_mvp` project. It categorizes them into allowed, partial, or prohibited based on the experimental evidence gathered during this phase.

| Claim | Status | Evidence Support | Risk / Limitation | How to Demonstrate / Verify Better |
| :--- | :---: | :--- | :--- | :--- |
| **“MVP técnico de risk scoring industrial.”** | **Permitido** | Evaluated on NASA CNC milling data across 3 seeds. Clean data loader, leakage reports, and calibration audits are implemented. | Imbalanced training, limited historical failure records in other datasets. | Standardize testing on a broader suite of industrial failure datasets (e.g. C-MAPSS). |
| **“Metadata/cycle son baselines fuertes en CNC.”** | **Permitido** | Under the operational protocol, a simple metadata-only logistic regression baseline achieves a mean AUPRC of up to $\sim 0.838$, dominating grid metrics. | This baseline is domain-dependent and relies on highly linear wear proxies. | Run benchmarks on datasets with randomized cutting cycles or variable durations. |
| **“Las señales sensoriales aportan valor en no-cycle.”** | **Permitido** | In the `no_cycle` split, pure metadata has AUPRC $\sim 0.2386$. Adding raw sensors increases this to $\sim 0.5200$ (delta $+0.2814$), showing physical signals carry wear information. | Higher variance in performance across seeds. | Implement robust feature selection methods to isolate key frequency bins. |
| **“Hardness/material-regime muestra valor sensorial.”** | **Permitido** | Under the `held_out_hardness_bin` split, metadata-only scores $0.5128$ AUPRC, while engineered features reach $0.7631$, proving sensors generalize well across material regimes. | Modest performance gain when combining features. | Scale tests on datasets with varying hardness levels. |
| **“Visual-JEPA global actual es MVP parcial y requiere rediseño dense.”** | **Permitido** | Global image AUROC on MVTec bottle ($\sim 0.67$) is beaten by the simple pixel-stat baseline ($\sim 0.80$). Heatmaps are coarse. | The global encoder fails on localized defect inspection. | Re-evaluate using a transformer encoder, block masks, and PatchCore-style KNN. |
| **“JEPA/future features aportan valor incremental.”** | **Parcial** | Standalone `predicted_future_z` reaches $0.5502$ AUPRC in the `no_cycle` split (delta $+0.3115$ over metadata), but is not stable across all splits. | High variance and modest improvement when added to engineered features. | Test on longer forecast horizons and larger datasets. |
| **“World model ayuda como feature complementaria.”** | **Parcial** | Predicted future embeddings conditional on actions contain failure risk cues but do not consistently beat current embeddings or no-action baselines. | Conditioning actions are low-dimensional and may not capture full dynamics. | Expand action vector dimensionality and use recurrent models. |
| **“DenseSensorJEPA puede ser útil tras rediseño.”** | **Parcial** | Implementation scaffold and pretraining tests pass, but the current direct pool AUPRC ($0.2002$) is very low. | Needs custom masked-to-visible token losses and better scoring mechanics. | Re-run with distance-weighted target loss. |
| **“SOTA (State of the Art).”** | **Prohibido** | The adversarial grid benchmark demonstrates that metadata proxies explain most of the performance, and official baselines are missing. | Making SOTA claims without standard, published evaluations is academically invalid. | Install official `sktime`/`aeon` and run official comparative runs. |
| **“JEPA supera de forma robusta a engineered features.”** | **Prohibido** | In matched no-cycle runs, standalone `predicted_future_z` has $0.5502$ mean AUPRC vs `sensor_engineered` at $0.5583$, indicating no robust superiority. | Latent features are noisy and often fail to outperform physical indicators. | Pretrain on larger scale datasets to improve self-supervision. |
| **“World model aprende dinámica causal.”** | **Prohibido** | The action-conditioned world model only learns correlation distributions, not structural causal mechanisms. | Confounding variables are highly prevalent in the CNC dataset. | Evaluate on physical simulators with active intervention. |
| **“Visual-JEPA compite con DINO/PatchCore.”** | **Prohibido** | The current global representation fails on local anomalies and is not compared against dense models. | Global representations cannot compete with dense patch indexing. | Run actual benchmark using frozen DINO feature cores. |
| **“Sistema listo para producción industrial sin piloto.”** | **Prohibido** | High class imbalance, dependency on specific tools, and lack of visual precision make deployment risky without local calibration. | The system could generate high false alarms in production. | Run a structured 4-6 weeks pilot before operational deployment. |
